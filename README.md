# Deposit Match $

Сервис для подбора банковских вкладов и расчета итоговой доходности.

Позволяет сравнивать предложения разных банков по условиям, фильтровать их и рассчитывать прибыль с учетом капитализации, бонусов и пользовательских параметров.

---

## Основные возможности

- Подбор вкладов по параметрам:
  - сумма
  - срок
  - валюта
  - пополнение / частичное снятие
  - капитализация
  - автопролонгация
-  Умная сортировка вкладов по доходности
-  Калькулятор доходности
-  Учет бонусов (зарплатный клиент, подписка и т.д.)
-  Поддержка нескольких банков
-  FastAPI backend + Vue 3 frontend

## Как работает расчет

1. Поиск подходящей базовой ставки
2. Применение бонусов
3. Расчет процентов:
   - с капитализацией
   - без капитализации
4. Вычисление итоговой суммы
